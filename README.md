
# Piernik

Gingerbread house lights and effects controller with Wi-Fi.

Software for Raspberry Pi Pico-W project (rather silly one).


## Hardware

 - Raspberry Pi Pico-W;
 - DFPlayer mini (with SD card);
 - 4 warm white LEDs;
 - 3 RGB LEDs;
 - small 8 Ohm speaker;
 - light sensor (photoresistor);
 - transistors, resistors, etc.


## Features

 - HTTP server - control effects using smartphone/laptop via web browser;
 - light effects - lights on/off, disco lights;
 - sound effects - playing mp3s from SD card;
